*Under heavy development yet!*

Installation
---

Install a recent version of *Docker* and *Docker Compose*

```bash
npm install -g lager
```

Create a new project
---

```bash
lager new <project-name>
```

Deploy on AWS
---

```bash
lager deploy
```
